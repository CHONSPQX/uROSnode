Creates a thread with a static stack.

Creates a new thread. The memory chunk for the stack is externally declared, and is simply referenced. It is usually allocated as a static buffer.

Precondition
The stack is big enough to avoid stack overflow.
Parameters
[out]idpPointer to the UrosThreadId where the created thread id is stored.
[in]namepPointer to the default thread name, valid for the whole the thread life. Null-terminated string.
[in]priorityThread priority.
[in]routineThread routine, entry point of the thread program.
[in]argpArgument passed to the thread routine. Can be NULL.
[in]stackpPointer to the externally allocated stack buffer.
[in]stacksizeStack size, in bytes.
Returns
Error code.
