void urosRpcParamClean ( UrosRpcParam *  paramp,
uros_bool_t  deep 
)

Cleans a XMLRPC parameter.

Deallocates any memory chunks allocated by the parameter, and invalidates it.

Precondition
The parameter is initialized.
The parameter data must have been allocated with urosAlloc().
Postcondition
paramp points to an invalidated UrosRpcParam object.
The parameter class is unchanged.
If desidred so, private members are deallocated. They must have been allocated with urosAlloc().
Parameters
[in,out]parampPointer to an initialized UrosRpcParam object.
[in]deepPerforms deallocation of private members.

UrosRpcParamNode *  nodep

Removes a node from the XMLRPC parameter list.

The node, if found, is unlinked from the list and returned.

Postcondition
The node does not belong to the list.
The node has no successors.
The list length is decremented by one.
The list is consistent.
Parameters
[in,out]listpPointer to an initialized UrosRpcParamList object.
[in]nodepPointer to the node to be removed from the list.
Returns
Pointer to the removed (unlinked) node.
Return values
nodepThe addressed node, if found inside the list.
NULLNo such node found inside the list.
