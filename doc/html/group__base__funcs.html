int urosStringCmp ( const UrosString *  str1,
const UrosString *  str2 
)

Compares two string objects.

This function starts comparing the first character of each string. If they are equal to each other, it continues with the following pairs until the characters differ or until there are no more characters in one string.

Note
This is functionally equivalent to strcmp(), but involves UrosString objects instead of null-terminated strings.
Two empty strings are equal.
If a string is empty but the other is not, then the first character is the returned value (with the appropriate sign).
Parameters
[in]str1Pointer to the first UrosString operand, valid string.
[in]str2Pointer to the second UrosString operand, valid string.
Returns
Integral value indicating the relationship between the strings.
Return values
0Both strings are equal.
positiveThe first character that does not match has a greater value in str1 than in str2.
negativeThe first character that does not match has a lesser value in str1 than in str2.
