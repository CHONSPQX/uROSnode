void urosRpcParserObjectInit ( UrosRpcParser *  pp,
UrosConn *  csp,
char *  rdbufp,
size_t  rdbuflen 
)

Initializes the XMLRPC parser object.

Assigns the connection and the read buffer.

Precondition
The parser is not initialized.
Postcondition
The parser is assigned a connection and an optional temporary buffer. The other fields are reset.
Parameters
[in,out]ppPointer to an allocated UrosRpcParser object.
[in]cspPointer to an UrosConn connection from which to receive the stream to be parsed.
[in]rdbufpPointer to the temporary buffer, used to parse strings ahead. Can be NULL only if rdbuflen is 0.
[in]rdbuflenLength of the temporary read buffer.

uros_err_t urosRpcParserHttpRequest ( UrosRpcParser *  pp)

Parses an incoming HTTP header section for an XMLRPC request.

Receives the HTTP header section of a POST request at the root, which is in the form: 

POST /RPC2 HTTP/1.(0|1)

The required headers are:

  • Content-Length, which is used by the parser to receive the correct number of pending characters,
  • Content-Type, which must be text/xml. Any other header is simply ignored.
Parameters
[in,out]ppPointer to an initialized UrosRpcParser object.
Returns
Error code.
