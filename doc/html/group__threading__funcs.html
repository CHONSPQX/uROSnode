uros_err_t urosThreadPoolWorkerThread ( UrosThreadPool *  poolp)

Worker thread.

This is the actual thread function created by urosThreadPoolCreateAll(). After creation, it waits until the user starts this thread by calling urosThreadPoolStartWorker(), when this is the first ready thread selected.

After executing the routine defined with urosThreadPoolObjectInit(), this thread goes back into the waiting state.

To exit from this thread, it first has to be put back into the waiting state. Then, the exitFlag of the pool must be set to true, so that by calling urosThreadPoolStartWorker() again on this thread, the latter knows it must exit and not execute the user routine.

Parameters
[in]poolpPointer to the UrosThreadPool which created this thread.
Returns
Error code returned by the user routine.
