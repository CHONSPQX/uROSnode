Initializes a node object.

This procedure:

  • initializes the status locks
  • initializes and allocates memory pools

Precondition

Find new publishers for a given topic.

Scans through the provided publishers list to look for any new publishers.

Parameters
[in]topicnamepPointer to a non-empty string which names the targeted topic.
[in]publisherspPointer to an UrosRpcParam with UROS_RPCP_ARRAY class. It contains the list of current publishers, received for example through a XMLRPC call to subscribeTopic(). Each publisher is addressed by its URI.
[out]newpubspPointer to an empty list which will be populated by the newly discovered publishers, if any.
Returns
Error code.
