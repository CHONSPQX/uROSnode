#define urosAssert (   expr)    UROS_NOP

Evaluates an assertion.

If the constraint expression does not hold, the system is halted.

Warning
An assertion is a very strong assumption. If an assertion does not hold, the whole system is halted. This is why it is commonly used only for very strict constraints, which must always hold.
Note
By default it does nothing. The user must provide an urosAssert() macro inside the urosconf.h configuration file to achieve the required behavior with platform-dependent code.
Assertions can be disabled inside the urosconf.h configuration file. A global switch is UROS_USE_ASSERT.
Parameters
[in]exprAssertion constraint expression, which must hold.
Example
#include <assert.h>
#define urosAssert(expr) \
assert(expr)
void load_value(int *resultp) {
urosAssert(resultp != NULL);
*resultp = 123;
}

#define urosError (   when,
  action,
  msgargs 
)    { if (when) { action; } }

Checks if an error occurred, and generates a message.

If the error condition holds, a formatted error message is generated, and a statement (even a code block) is executed.

Note
By default it simply ignores the formatted message. The user must provide an urosError() macro inside the urosconf.h configuration file to achieve the required behavior with platform-dependent code.
Error messages can be disabled inside the urosconf.h configuration file. A global switch is UROS_USE_ERROR_MSG.
Parameters
[in]whenError condition, which should not hold for a correct code.
[in]actionA statement, or code block, which is executed if the error condition holds.
[in]msgargsError message arguments. Must be enclosed between round parentheses in a printf() compatible format.
Example
#include <urosUser.h>
#define urosError(when, action, msgargs) \
{ if (when) { \
urosUserErrPrintf("Error at %s:%d\n" \
" function: %s\n" \
" reason: %s\n" \
" message: ", \
__FILE__, __LINE__, \
__PRETTY_FUNCTION__, \
#when); \
urosUserErrPrintf msgargs ; \
{ action; } } }
UrosString globalPassword = { 0, NULL };
uros_bool_t set_password(const UrosString *passwp) {

UROS_STRARG(passwp), passwp->length));
urosStringClean(&globalPassword);

}

urosAlloc()
Parameters

Returns

Return values

Example
