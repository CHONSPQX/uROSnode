uros_err_t urosRpcStreamerXmlEndHack ( UrosRpcStreamer *  sp)

Hacks the XMLRPC message length.

This function is called to end a XMLRPC message. It behaves in two ways, depending on the status of the write buffer.

  • If the write buffer of the UrosRpcStream was flushed since the beginning of the message, then a sequence of LF ('\n') is appended to match the UROS_RPCSTREAMER_FIXLEN body length.
  • If the write buffer was not flushed, then it is possible to overwrite the Content-Length. It is adjusted so that it matches the current body length.
    See Also
    UROS_RPCSTREAMER_FIXLEN
    urosRpcStreamerHttpContentLength()
    Precondition
    No XMLRPC message can exceed UROS_RPCSTREAMER_FIXLEN.
    Parameters
    [in,out]spPointer to an initialized UrosRpcStreamer object.
    Returns
    Error code.
