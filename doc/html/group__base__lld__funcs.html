void* uros_lld_alloc ( size_t  size)

Allocates a memory block.

This function tries to allocate a memory block of the required size inside the default heap of the operating system.

See Also
urosAlloc()
Precondition
There is enough contiguous free space inside the default heap.
Parameters
[in]sizeSize of the memory block to be allocated, in bytes.
Returns
The address of the allocated memory chunk.
