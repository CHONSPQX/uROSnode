uros_err_t urosRpcCallSearchParam ( const UrosAddr *  addrp,
const UrosString *  caller_id,
const UrosString *  key,
UrosRpcResponse *  resp 
)

Search for parameter key on the Parameter Server.

Search starts in caller's namespace and proceeds upwards through parent namespaces until Parameter Server finds a matching key.

Its behavior is to search for the first partial match. For example, imagine that there are two robot_description parameters:

  • /robot_description
    • /arm
    • /base
  • /pr2
    • /robot_description
      • /base

If starting in the namespace /pr2/foo and search for robot_description, searchParam() will match /pr2/robot_description. If searching for robot_description/arm it will return /pr2/robot_description/arm, even though that parameter does not exist (yet).

See Also
http://www.ros.org/wiki/ROS/Parameter%20Server%20API
Parameters
[in]addrpPointer to the Master API connection address.
[in]caller_idROS caller ID. Non-empty string pointer.
[in]keyParameter name to search for. Non-empty string pointer.
[out]respPointer to the UrosRpcResponse response object. Format: 
[ int code, str statusMessage, str foundKey ]
Returns
Error code.
